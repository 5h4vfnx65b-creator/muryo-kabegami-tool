# 無料壁紙ツール MVP (Static)

写真を「切らずに」壁紙サイズへ最適化する、超シンプルなMVPです。
- 生成は **ブラウザ内（Canvas）** で完結します（サーバー不要）
- 余白部分は **ぼかし背景で自然に埋める** 方式（MVPとして安定）

## 使い方（ローカル）
1. このフォルダを開く
2. `index.html` をブラウザで開く  
   - Chrome / Safari でOK

※ ちゃんと動かない場合はローカルサーバー推奨：
- macOS: `python3 -m http.server 5173` → `http://localhost:5173`

## デプロイ（無料でOK）
- GitHub Pages / Netlify / Cloudflare Pages / Vercel で **静的サイト** としてそのまま出せます。

## 広告の入れ方
`index.html` 内の `ad-slot` を、AdSenseや他広告タグに差し替えます。
- ボタン付近に広告を寄せすぎない（誤タップ対策）
- ダウンロード前は「リワード型」っぽいUXにしてます（今はプレースホルダー）

## 機種の追加
`presets.js` に `{ name, w, h }` を足せばOK。


## 仕上げモード
- くっきり（ぼかし無し）: 端のボケを減らします（背景は写真を拡大して使います）
- 自然（ぼかし）: 余白をぼかしで自然にします
- 背景をそのまま拡大（ぼかし無し）: 背景もくっきり（背景はクロップされます）

※ 周りの風景を“生成”して自然に拡張する場合は、サーバー側で画像拡張（outpainting）APIを呼ぶ構成が必要です。
